Taidot

Tässä on lista tekniikoista, joita olen oppinut opintojeni aikana käyttämään.

  • Java
  • JavaScript
  • HTML/CSS
  • React
  • Python
  • C#
  • Swift
  • Kotlin
  • Pug

Hallitsen parhaiten näistä tekniikoista kolme ensimmäistä. Olen aloittanut ohjelmoinnin Javasta ja käynyt siitä myös olio-ohjelmointi kurssin. Javan kanssa oppimia taitojani olen hyödyntänyt C#-kielen sekä Kotlinin opiskelussa. C#-kieltä käytimme koulumme peliprojektissa ja Kotlinia Android-sovelluksen rakentamisessa. Vastaavanlaisen iOS-sovelluksen kehittämiseen käytimme Swift-kieltä. Näiden sovellusprojektien repositoriot löydät minun Github sivulta.

Tämän verkkosivun omistaa ja on luonut Kaisa Hakola. Ota yhteyttä, mikäli kiinnostuit töistäni. Päivitetty 01/2024.
